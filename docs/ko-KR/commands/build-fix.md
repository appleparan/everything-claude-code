# 빌드 및 수정

TypeScript 및 빌드 오류를 점진적으로 수정합니다:

1. 빌드 실행: npm run build 또는 pnpm build

2. 오류 출력 분석:
   - 파일별로 그룹화
   - 심각도별로 정렬

3. 각 오류에 대해:
   - 오류 컨텍스트 표시 (전후 5줄)
   - 문제 설명
   - 수정 방안 제시
   - 수정 적용
   - 빌드 재실행
   - 오류 해결 확인

4. 중지 조건:
   - 수정이 새로운 오류를 도입한 경우
   - 3회 시도 후에도 동일한 오류가 지속되는 경우
   - 사용자가 중지를 요청한 경우

5. 요약 표시:
   - 수정된 오류
   - 남은 오류
   - 새로 도입된 오류

안전을 위해 한 번에 하나의 오류만 수정하세요!
